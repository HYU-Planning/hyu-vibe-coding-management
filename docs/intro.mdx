# 소개

한양대학교 **바이브 코딩(Vibe Coding) 방법론** 교육 자료입니다.

이 사이트는 대학 구성원을 대상으로 바이브 코딩 방법론을 교육하기 위한 자료를 체계적으로 정리하고 공유하기 위해 만들어졌습니다.

## 이 자료의 목적

- 바이브 코딩 방법론의 핵심 개념과 원칙을 정리
- 실습 예제와 사례를 통한 학습 지원
- 강의/워크숍에서 활용할 수 있는 표준 교재 제공

## 시작하기

왼쪽 사이드바에서 관심 있는 주제를 선택해 학습을 시작하세요. 목차는 `docs/` 폴더 구조를 기반으로 자동 생성됩니다.

## 기여 방법

이 저장소는 GitHub를 통해 관리됩니다. 자료를 추가하거나 수정하려면:

1. 저장소를 clone 합니다.
2. `docs/` 폴더 아래에 마크다운(`.md`/`.mdx`) 파일을 추가하거나 수정합니다.
3. `main` 브랜치에 push 하면 GitHub Actions가 자동으로 사이트를 다시 빌드하고 배포합니다.
